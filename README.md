# Face Detection & Classification

A Python pipeline for real-time face detection and identity classification using OpenCV and TensorFlow/Keras.

## Overview

This project provides a complete workflow: capture images, detect faces, prepare a cropped face dataset, train a CNN classifier, and run live face recognition from a webcam.

### Architecture

```
raw_dataset/          →  capture_image.py  →  dataset_faces/  →  train_classifier.py  →  face_classifier.keras
(class images)           (capture webcam)       (cropped faces)    (fine-tune MobileNetV2)      (trained model)
                                                                                                    ↓
                                                          detect_and_classify.py ← webcam feed
                                                          (detect + classify in real-time)
```

## Files

| File | Description |
|------|-------------|
| `capture_image.py` | Captures a single photo from the webcam and saves it to disk. |
| `face_detection.py` | Detects faces using OpenCV Haar cascades. Works on images or webcam. |
| `prepare_faces.py` | Prepares a cropped face dataset using Haar cascade detection. |
| `prepare_faces_cnn.py` | Prepares a cropped face dataset using a heavier but more accurate DNN (SSD Caffe) detector. |
| `train_classifier.py` | Trains a MobileNetV2-based classifier on the prepared dataset. |
| `detect_and_classify.py` | Runs live face detection + classification from the webcam. |
| `class_names.json` | Maps class indices to label names (generated by `train_classifier.py`). |
| `face_classifier.keras` | The trained classifier model (generated by `train_classifier.py`). |
| `models/` | Contains the SSD Caffe face detection model (`deploy.prototxt` + `.caffemodel`). |

## Prerequisites

- Python 3.8+
- `opencv-python`

```bash
pip install opencv-python tensorflow numpy
```

## Usage

### 1. Capture Images

Capture photos from the webcam:

```bash
python capture_image.py
```

- Press **`s`** to save a photo.
- Press **`q`** to quit.

### 2. Prepare Dataset

Organize your captured images into class folders inside `raw_dataset/`:

```
raw_dataset/
├── name/
│   ├── photo1.jpg
│   └── photo2.jpg
└── unknown/
    ├── photo3.jpg
    └── photo4.jpg
```

Then crop faces from the images. Two detectors are available:

**Haar Cascade (fast, lighter):**

```bash
python prepare_faces.py
```

**DNN SSD (slower, more accurate):**

```bash
python prepare_faces_cnn.py \
    --input raw_dataset \
    --output dataset_faces \
    --confidence 0.55 \
    --padding 0.25 \
    --output-size 160x160
```

Options for `prepare_faces_cnn.py`:

| Argument | Default | Description |
|----------|---------|-------------|
| `--input` | `raw_dataset` | Input folder with class subfolders |
| `--output` | `dataset_faces` | Output folder for cropped faces |
| `--model-dir` | `models` | Folder for the DNN model files |
| `--confidence` | `0.55` | Minimum detection confidence |
| `--padding` | `0.25` | Padding around detected face |
| `--output-size` | `160x160` | Resize cropped faces (use `none` to keep original) |
| `--save-all-faces` | — | Save every detected face, not just the largest |

### 3. Train the Classifier

```bash
python train_classifier.py
```

This trains a MobileNetV2 model (fine-tuned from ImageNet weights) on the cropped faces in `dataset_faces/`. Outputs:

- `face_classifier.keras` — the trained model
- `class_names.json` — class name mapping

Key training parameters (edit in the script):

| Parameter | Default | Description |
|-----------|---------|-------------|
| `EPOCHS` | 12 | Max training epochs |
| `BATCH_SIZE` | 32 | Batch size |
| `VALIDATION_SPLIT` | 0.2 | Fraction used for validation |
| `learning_rate` | 0.0005 | Optimizer learning rate |

Early stopping with `patience=4` is applied — training halts if validation accuracy doesn't improve.

### 4. Live Detection & Classification

Run real-time face detection and classification from the webcam:

```bash
python detect_and_classify.py --camera 0 --confidence 0.70
```

Options:

| Argument | Default | Description |
|----------|---------|-------------|
| `--camera` | `0` | Webcam index |
| `--confidence` | `0.70` | Minimum confidence to display a class label (below this shows "unknown") |

## Detection Methods

This project uses two face detection approaches:

- **Haar Cascade** (`face_detection.py`, `prepare_faces.py`): Fast, CPU-only, good for real-time. Used by default.
- **DNN SSD Caffe** (`prepare_faces_cnn.py`): Heavier but more accurate. Automatically downloads model files to `models/` on first run.
